The following things are currently supported in the input:
* basic term rewriting systems (coco format)
* basic term rewriting systems (Cora format)
* many-sorted term rewriting systems (coco format)
* many-sorted term rewriting systems (Cora format)
* applicative higher-order term rewriting systems (Cora format)

The following should all eventually be handled in the input (but are not now):

* integers, booleans and calculation
* integer and boolean constraints
* general SMT support
* support for higher-order rewriting
* confluence analysis
* termination analysis
* equivalence analysis
* Java parser
